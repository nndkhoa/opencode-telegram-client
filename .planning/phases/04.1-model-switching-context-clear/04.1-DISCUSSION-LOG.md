## `/model <name>` — Global vs per-session

| Option | Description | Selected |
|--------|-------------|----------|
| Global (`PATCH /config`) | Applies to all sessions — matches OpenCode config API shape | ✓ |
| Per-session | Would require tracking model per chat ID locally | |

**User's choice:** Global — `PATCH /config`.
