# Quick task 260329-h5o: Markdown tables as raw for Telegram

## Goal

GFM pipe tables were converted to HTML `<table>`, then stripped by `sanitize-html` (not in allowlist). Telegram Bot HTML also does not render tables. Show the aligned pipe markdown inside a fenced block so it becomes `<pre><code>` (monospace).

## Tasks

1. **Preprocess**: Before `marked()`, wrap each `table` token’s `raw` source in a fenced code block via `lexer` + `walkTokens`.
2. **Test**: Assert output contains `<pre>` and pipe text, not `<table>`.
